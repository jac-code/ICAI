Se entregan un total de x8 archivos.

- x1 con nombre primeros_apartados.ipynb que como indica su nombre tiene los apartados 1.1, 1.2, 1.3, 1.4 y 1.5 de la práctica
- x1 block.py --> es la clase bloque que será llamada como un módulo desde la clase blockchain
- x1 blockchain.py --> es la clase blockchain que se llamará como otro módulo desde la clase node
- x1 node-py --> la clase node simula un nodo de la red 
- x3 node_x.ipynb --> estos tres archivos son unos notebooks que al ejecutarlos cada uno levanta un nodo de la red
		      los puertos son --> nodo 1 = 8085, nodo 2 = 8086, nodo 3 = 8087
- x1 testing.ipynb --> es un notebook donde se simula todas las acciones que se pueden hacer en una red blockchain, desde añadir nodos a la red,
		       minar nuevos bloques etc...

MODO DE EJECUCIÓN

- Para que todo funcione correctamente se debe abrir los archivos node_1.ipynb, node_2.ipynb y node_3.ipynb para levantar los nodos con los que se va a trabajar 

- Una vez levantados los nodos se puede abrir el archivo de testing.ipynb 

- El archivo testing.ipynb simula todas las acciones que se pueden hacer con la red blockchain

- El archivo testing.ipynb tiene explicaciones y los pasos a seguir para poder hacer todas las pruebas

IMPORTANTE QUE TODOS LOS ARCHIVOS ESTÉN EN LA MISMA CARPETA PARA QUE LOS import FUNCIONEN CORRECTAMENTE
